# UndoBoss - tShock Plugin

## What is UndoBoss
Basically a plugin to toggle whether a boss has been defeated

## How to use it
Use /undoboss (/uboss) and a boss name/alias for the second parameter to toggle
eg. ` /uboss kingslime `
Command Permission: (undoboss.use)
